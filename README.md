# Cat Api Test

A small app to display cats from:
https://thecatapi.com/

##### Running Locally
1. Acquire a api key from https://thecatapi.com/
2. Create a .env file with entries API_KEY and PORT
3. Create a environment variable in your terminal ENV and set it's value to DEV
4. Run command npm i
5. Run command npm run build
6. Run command npm start
7. The website/server will now be live at the port you specified in step 2

##### Live

A live version can be found here:

https://ziya-emanet-cat-api-test.herokuapp.com/
